## 🚀 使用方法

### 编译项目

```bash
cd ~/Github/ziv-lang
cargo build --release
```

### 运行示例

```bash
# Hello World
./target/release/ziv examples/stdlib/hello.ziv -o hello
./hello

# 数学函数
./target/release/ziv examples/stdlib/math_demo.ziv -o math_demo
./math_demo
```

### 运行测试

```bash
cargo test
```
